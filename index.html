内容

这里是 Bash 相关的、4篇质量极高的学习指南。分别为：

  1. 开篇：Bash 是什么，它活在哪里？

    简介 Bash，以及如何安装和启动它；终端，键盘及结果显示；程序，进程以及他们内部流动的信息如何被连接在一起

  2. 命令与参数：我要如何给 bash 下指令?

    关于什么是命令，如何发布他们；交互模式与脚本；命令语法，通过名称搜索命令与程序；参数与单词分割，以及输入和输出重定向

  3. 变量与扩展：我如何储存并使用数据？

    Bash 参数与变量；环境变量，特殊参数以及数组参数；扩展参数，扩展运算控制符，命令替换与进程替换；路径名扩展，尾数扩展与括号扩展

  4. 测试与条件：不同的命令给不同的数据

    退出代码，成功与失败，测试文件，字符串与数值，处理不同的条件，条件操作符以及条件复合命令

  • 练习题索引

按原作者的构想，这份 Bash 指南共11章内容。遗憾的是，作者只完成了前面4章，但因其写作质量之高、讲解之透彻，还是极为可贵，遂将这4篇指南翻译并分享于此。

原作者对整份指南的介绍如下：

这份指南是对bash shell 基础及高级概念的介绍。
旨在把编写强力bash脚本以及高效快速交互式运用shell的最好、最安全、最可靠的方法，教给bash新手及长时间使用者。
这份指南已经过多版迭代，是就此话题无论在互联网上、公共图书馆还是书店，都始终严重缺乏持续、清晰又可靠的语言的结果。

如此精彩的教程却不完整，实在可惜。但此前，作者与GreyCat其实已就 Bash 完整 写过一份旧版指南，感兴趣的读者请移步此处查看，或在线阅读pdf版。
